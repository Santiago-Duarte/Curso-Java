# Java Course: MoureDev (9h) - Fundamentos y POO
[![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)](https://www.oracle.com/java/)

Este repositorio contiene el desarrollo práctico y la implementación de conceptos del curso de Java de MoureDev. El enfoque se centró en la transición de la sintaxis básica hacia la arquitectura orientada a objetos.

## 🛠️ Stack Tecnológico
* **Lenguaje:** Java 17+
* **IDE:** IntelliJ IDEA
* **Control de Versiones:** Git / GitHub

## 📚 Contenido Implementado

### Fundamentos y Estructuras
- **Variables y Operadores:** Tipado fuerte y manejo de memoria.
- **Control de Flujo:** Estructuras condicionales y bucles optimizados.
- **Estructuras de Datos:** Manejo de `Arrays`, `ArrayList`, `Map` y `Set`.

### Programación Orientada a Objetos (POO)
- **Abstracción y Encapsulamiento:** Uso de modificadores de acceso.
- **Herencia y Polimorfismo:** Creación de jerarquías de clases y sobrescritura de métodos.
- **Interfaces:** Definición de contratos de comportamiento.

### Manejo de Errores
- Gestión de excepciones con bloques `try-catch` para un código robusto.

## 🚀 Ejecución
Cada módulo se encuentra en el directorio `src/`. Para ejecutar cualquier clase con un método `main`:
1. Abrir en IntelliJ IDEA.
2. Navegar hasta la clase deseada.
3. Presionar `Shift + F10` o clic en el icono verde de "Run".
